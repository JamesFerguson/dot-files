# schedule

A little repo to make faffing with cronjobs nicer and git tracked.

# Installation

 1. Clone
 1. `cd schedule`
 1. `gem install bundler`
 1. `bundle`
 1. `whenever -i`
 
# Updating crontab

`whenever -i` - uses absolute path and comments to track block in crontab and only change those lines
